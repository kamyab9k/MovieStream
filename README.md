# registerPage

A **Register** screen and a **UserInfo** screen.

 When filled all the blanks and clicked register, user is navigated to UserInfo screen.

Techonologies used in this project :

* Jetpack compose
* Shared preferences
* MVVM
* User session management
* Navigation compose
* Repository pattern

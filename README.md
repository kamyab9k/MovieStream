# MovieCatalog
a jetpack compose movie Streaming app that recommends latest **popular** movies and series.User data is permenant and stored in shared pref.

Techonologies used in this project :

* Jetpack compose
* user session management
* shared prefrences
* Restful api
* MVVM
* Retrofit
